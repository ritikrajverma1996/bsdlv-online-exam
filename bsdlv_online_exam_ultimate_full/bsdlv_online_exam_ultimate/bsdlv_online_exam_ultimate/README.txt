BSDLV Online Exam - Ultimate Web App

Features:
- Admin and Student login
- Hindi/English language switch
- Students add/import from Excel
- Student photo upload
- Questions add/edit/delete/import from Excel
- Exam create/edit/delete with manual question picker
- Class-wise and section-wise exam assignment
- Exam instructions page
- Timer + auto submit
- One student one attempt
- Admit card PDF
- Result card PDF
- Logo upload
- Mobile friendly UI

Default admin:
username: admin
password: admin123

Run locally:
1. Extract ZIP
2. Run run_local.bat
3. Open http://127.0.0.1:5000

Excel headers:
Students: student_id, name, student_class, section, password
Questions: question, optionA, optionB, optionC, optionD, correctAnswer, marks, difficulty, subject, chapter
